# Privacy Notice — finanzas_felix

**Last updated:** 24 April 2026

## About this application

`finanzas_felix` is a personal, non-commercial application developed by Félix Méndez (fxmcan@gmail.com) to track his own personal bank accounts via the Enable Banking API in restricted production mode.

## Data controller

Félix Méndez
Email: fxmcan@gmail.com

## What data is processed

This application processes exclusively **the data controller's own banking data**, retrieved via PSD2 account information services from his own Openbank accounts under his explicit SCA consent.

Data processed:

- Account identifiers (IBAN, internal UUIDs from the TPP)
- Transaction records (date, amount, counterparty, description)
- Account balances

**No data from third parties is processed.** This is a single-user personal finance tool.

## Purpose of processing

Personal budgeting and expense tracking. Data is consumed only by the data controller via private widgets and scripts on his own devices.

## Data storage and retention

- Data is stored exclusively on a private server (Hetzner VPS) located in the European Union, under the sole control of the data controller.
- No data is shared with third parties beyond the technical transit through Enable Banking (acting as the TPP).
- Data is retained indefinitely for historical analysis. The data controller can delete the database at any time.

## Legal basis

GDPR Art. 6(1)(a) — consent given by the data controller (who is also the data subject) through SCA authorization with his own bank.

## Rights

As both data controller and sole data subject, Félix Méndez retains full access to, rectification of, and deletion of all processed data at any moment.

## Third-party processors

- **Enable Banking Oy** (Finland) — TPP / AISP providing PSD2 access to the bank. See: https://enablebanking.com/privacy-notice/
- **Openbank S.A.** (Spain) — the ASPSP providing account data via PSD2.
- **Hetzner Online GmbH** (Germany) — infrastructure provider for the VPS where data is stored.
